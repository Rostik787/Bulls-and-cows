## 📌 Про проєкт

Telegram-бот класичної гри **«Бики та Корови»**.

Бот загадує 4-значне число з унікальних цифр, а гравець намагається його відгадати.

Проєкт створений як:
- навчальний приклад
- демонстрація async Python
- робота з Telegram Bot API та SQLite

---

## 🎮 Як грати

- 🐂 **Бик** — цифра на правильній позиції
- 🐄 **Корова** — цифра є, але не на своєму місці

Гравець вводить числа через inline-клавіатуру та отримує підказки.
